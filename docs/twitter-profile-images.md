## How to Add New Images

Run the backfill script to download profile images for any new accounts:

```bash
node scripts/backfill-twitter-profile-images.cjs
```

To see a report of current status:

```bash
node scripts/backfill-twitter-profile-images.cjs --report
```

Images are stored in `/public/twitter-profiles/` and served locally for fast loading.

## Technical Details

- **Source:** [Unavatar.io](https://unavatar.io) (free service)
- **Fallback:** UI Avatars (initials with orange background)
- **Storage:** `/public/twitter-profiles/{handle}.jpg`
- **Component:** `src/components/dashboard/components/twitter-profile-avatar.tsx`
