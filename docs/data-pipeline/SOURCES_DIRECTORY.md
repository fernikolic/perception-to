## Adding New Sources

To add a new source:

1. **Determine the category** - Which of the 9 categories does it fit?
2. **Set up IFTTT/RSS** - Configure the feed in Feedly/IFTTT
3. **Update this document** - Add the feed to the appropriate section
4. **Update categorization code** - If needed, add to `src/lib/utils/outlet-categorization.ts`

See also:
