## Data Quality Notes

This list contains some variations in naming conventions:
- Some entries include job titles (e.g., "Senior Contributor", "Forbes Staff")
- Some entries are organization names rather than individual authors
- Some entries are email addresses (TV Azteca)
- Cointelegraph uses a standardized format: "Cointelegraph by [Author Name]"

**Query Used:**
```sql
SELECT DISTINCT author_name, Outlet
FROM `triple-upgrade-245423.btcp_main_dataset.all_channels_data`
WHERE author_name IS NOT NULL
ORDER BY Outlet, author_name
```
