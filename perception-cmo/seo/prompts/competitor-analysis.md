# Competitor Analysis Prompt Templates

## Full Competitor Analysis

```
Perform comprehensive SEO competitor analysis for Perception:

**Competitors to analyze:**
1. LunarCrush (lunarcrush.com) - Social & market intelligence
2. Santiment (santiment.net) - Behavioral analytics, on-chain + social
3. Glassnode (glassnode.com) - On-chain market intelligence
4. The TIE (thetie.io) - Institutional crypto data
5. Messari (messari.io) - Crypto research platform

**For each competitor, analyze:**

1. **Keyword Targeting**
   - What keywords do they rank for?
   - What's their primary positioning?
   - Which keywords overlap with Perception's targets?

2. **Content Strategy**
   - What types of pages do they have?
   - Do they have comparison pages?
   - Blog/resource section size and topics?
   - FAQ presence and topics?

3. **Technical SEO**
   - Title tag format
   - Meta description approach
   - Structured data usage
   - Site speed (if observable)

4. **Differentiation Opportunities**
   - Where can Perception differentiate?
   - Keyword gaps they don't target
   - Content gaps we can fill

**Output:**
- Competitor matrix comparison table
- Keyword opportunities (they rank, we don't)
- Content gaps to exploit
- Specific page recommendations
```

## SERP Analysis for Specific Keyword

```
Analyze the SERP for keyword: "[KEYWORD]"

1. **Search the keyword** using WebSearch tool

2. **For top 10 results, analyze:**
   - Domain and page URL
   - Title tag used
   - Meta description
   - Content type (article, product page, listicle)
   - Word count estimate
   - Key topics covered

3. **Identify patterns:**
   - Common title formats
   - Common content structures
   - Entities mentioned
   - User intent signals

4. **Assess Perception's opportunity:**
   - Current position (if any)
   - Content gap vs top results
   - Angle we could take
   - Specific recommendations

5. **Output:**
   - SERP overview table
   - Content brief for ranking
   - Title/description recommendations
   - Topic outline to beat current results
```

## Keyword Gap Analysis

```
Find keyword opportunities where competitors rank but Perception doesn't:

**Process:**

1. Use WebSearch to find competitor rankings for:
   - "crypto media monitoring" + competitor
   - "bitcoin sentiment analysis" + competitor
   - "crypto intelligence platform" + competitor
   - And other target keywords

2. Identify keywords where:
   - Competitors appear in top 10
   - Perception doesn't appear
   - Search intent matches our product

3. Categorize opportunities:
   - High priority: High intent, low competition
   - Medium priority: High intent, medium competition
   - Long-tail: Lower volume, easy wins

4. **Output:**
   - Prioritized keyword opportunity list
   - Estimated difficulty
   - Content type needed to rank
   - Quick win recommendations
```

## Comparison Page Research

```
Research to create comparison page: Perception vs [COMPETITOR]

1. **Analyze competitor's site:**
   - Key features they promote
   - Pricing structure
   - Target audience messaging
   - Unique selling points

2. **Search for existing comparisons:**
   - "[competitor] vs" searches
   - "[competitor] alternative" searches
   - Review sites covering the competitor

3. **Identify differentiation angles:**
   - Where Perception is genuinely better
   - Where competitor is stronger (be honest)
   - Different use cases for each

4. **User intent research:**
   - What are people looking for in this comparison?
   - What questions do they have?
   - What would convince them to try Perception?

5. **Output:**
   - Feature comparison table data
   - FAQ questions for schema
   - Content outline (2000+ words)
   - Title/meta description recommendations
   - CTA strategy
```

## Content Gap Analysis

```
Identify content gaps vs competitors:

1. **Analyze competitor content:**
   - Blog topics they cover
   - Resource/guide pages
   - Use-case pages
   - Educational content

2. **Compare to Perception's content:**
   - What topics do we not cover?
   - What formats are we missing?
   - Where is competitor content thin?

3. **Prioritize gaps by:**
   - Search volume potential
   - Relevance to our product
   - Competitive difficulty
   - Conversion potential

4. **Output:**
   - Content gap matrix
   - Prioritized content calendar
   - Brief for each recommended piece
   - Internal linking opportunities
```
