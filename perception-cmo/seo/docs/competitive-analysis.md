# Competitive Analysis for SEO

## Direct Competitors

### 1. LunarCrush (lunarcrush.com)

**Positioning:** Real-Time Social & Market Intelligence Powered by AI

**What They Do:**
- Social media analytics for crypto
- Sentiment tracking across Twitter, Reddit, etc.
- Trading signals based on social data
- API for algorithmic trading

**SEO Keywords They Rank For:**
- crypto sentiment analysis
- social media crypto tracking
- cryptocurrency social metrics
- crypto social intelligence

**Perception's Advantage:**
- We monitor 250+ sources vs their social-only focus
- We cover legacy media (CNBC, Bloomberg, etc.)
- We're built for PR workflows, not trading
- We provide citation-backed reports

**Comparison Page Angle:**
"Media intelligence vs social sentiment - Perception tracks the full media landscape while LunarCrush focuses on social buzz."

## Keyword Gap Analysis

### Keywords We Should Target (Competitors Rank, We Don't)

**High Priority:**
1. "crypto sentiment analysis tool" - Santiment, LunarCrush rank
2. "bitcoin media monitoring" - No clear winner, opportunity
3. "crypto narrative tracking" - CoinGecko, Dune rank
4. "crypto PR tools" - Mostly PR agencies rank
5. "bitcoin coverage tracking" - Open opportunity

**Medium Priority:**
1. "crypto intelligence platform" - Multiple competitors
2. "bitcoin news aggregator" - Many generic tools
3. "crypto media tracker" - Open opportunity
4. "cryptocurrency research platform" - Messari dominates

**Long-Tail Opportunities:**
1. "how to track bitcoin narratives"
2. "crypto PR monitoring best practices"
3. "bitcoin media sentiment analysis"
4. "crypto coverage analysis tool"
5. "track cryptocurrency news coverage"
