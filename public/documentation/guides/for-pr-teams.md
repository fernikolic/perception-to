# Guide for PR & Communications Teams

> Track coverage, identify journalists, and manage reputation

## Overview

PR and communications professionals need real-time awareness of media coverage, relationships with key journalists, and early warning of potential crises. Perception provides comprehensive media monitoring with sentiment analysis and outlet intelligence.

## Your Key Use Cases

### 1. Media Monitoring

**Goal:** Track all coverage of your organization or client.

**How to do it:**
1. Create a watchlist item with your company name and variations
2. Include executive names, product names, and common misspellings
3. Set up real-time alerts for immediate notification
4. Review weekly reports for comprehensive analysis

**Example watchlist setup:**
```
Name: [Your Company]
Search terms: "[Company], [CEO name], [product names], [ticker]"
```

### 2. Journalist Intelligence

**Goal:** Identify which journalists and outlets cover your space.

**How to do it:**
1. Search for your topic area
2. Note which outlets appear most frequently
3. Track individual journalist coverage patterns
4. Build a media list based on actual coverage data

**What to track:**
- Which outlets cover your competitors?
- Who breaks stories in your space first?
- What's the sentiment pattern by outlet?

### 3. Competitive Coverage Analysis

**Goal:** Understand how competitors are being covered vs. your organization.

**How to do it:**
1. Create separate watchlist items for each competitor
2. Compare mention volumes weekly
3. Analyze sentiment differences
4. Identify outlets that favor competitors

### 4. Crisis Monitoring

**Goal:** Get early warning of potential reputation issues.

## Best Practices

### Setting Up Alerts

| Alert Type | When to Use |
|------------|-------------|
| Real-time (All) | Active campaigns, crisis situations |
| Real-time (High only) | Routine monitoring |
| Weekly digest | Ongoing awareness |

### Search Term Strategy

**Comprehensive coverage:**
```
"[Company exact name]", "[Common abbreviation]",
"[CEO first last name]", "[Product names]", "[Ticker]"
```

**Exclude noise:**
If you share a name with common terms, be more specific:
```
"[Company] bitcoin", "[Company] crypto"
```

### Interpreting Sentiment

| Sentiment Pattern | What It Means | Action |
|-------------------|---------------|--------|
| Consistent neutral | Factual coverage | Standard PR opportunity |
| Shifting positive | Good momentum | Amplify and extend |
| Sudden negative | Potential issue | Investigate and prepare response |
| Mixed sentiment | Controversy | Monitor closely, prepare messaging |

### Weekly PR Planning

**Monday:**
1. Review weekly report from previous week
2. Identify coverage gaps and opportunities
3. Note journalists who covered your space

**Throughout Week:**
1. Monitor for pitch opportunities
2. Track campaign coverage if active
3. Watch competitor announcements

**Friday:**
1. Compile coverage summary
2. Identify next week's opportunities
3. Update media lists

## Generating PR Briefs

When you need a comprehensive coverage analysis, generate a brief:

1. Select your company or topic
2. Choose **"PR & Communications"** audience
3. Review the generated brief for:
   - Media landscape overview
   - Key journalist and outlet analysis
   - Message effectiveness assessment
   - Recommended PR actions

### What PR Briefs Include

- **Coverage Analysis**: Who's covering you and how
- **Sentiment Insights**: Tone patterns and shifts
- **Outlet Intelligence**: Which publications to prioritize
- **Narrative Assessment**: Are your messages landing?
- **Recommendations**: Specific PR actions to take

## Advanced Tips

### Building Media Relationships

1. Track which journalists consistently cover your beat
2. Note their typical sentiment (are they friendly or critical?)
3. Identify what angles interest specific outlets
4. Use coverage data to personalize outreach

### Campaign Tracking

1. Create a dedicated watchlist item for campaign keywords
2. Set a baseline before campaign launch
3. Monitor daily during active campaigns
4. Compare pre/post metrics for reporting

### Reputation Management

1. Set up alerts for executive names
2. Monitor industry-wide sentiment (canary in coal mine)
3. Track competitor crises (lessons learned)
4. Build response playbooks based on common coverage types
