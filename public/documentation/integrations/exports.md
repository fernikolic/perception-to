# Data Exports

> Download your intelligence data for external analysis

## Overview

Perception allows you to export data for use in your own tools, reports, and analysis workflows.

## What You Can Export

### Search Results
Export the results of any search query including:
- Title and content
- Date and source
- Sentiment classification
- Topic tags

### Watchlist Data
Export coverage data for your tracked topics:
- All matching data points
- Sentiment breakdown
- Source distribution

### Report Data
Export data from your weekly reports:
- Mention counts
- Week-over-week comparisons
- Outlet breakdowns

---

## Export Formats

| Format | Best For |
|--------|----------|
| **CSV** | Spreadsheet analysis (Excel, Google Sheets) |
| **JSON** | Developer integration, custom tools |

### From Search Results

1. Run your search query
2. Apply any filters (date, sentiment, outlet)
3. Click the **Export** button
4. Select your preferred format
5. Download the file

### From Watchlist

1. Navigate to a watchlist item
2. View the detailed coverage
3. Click **Export Data**
4. Choose format and date range

---

## Export Limits

To ensure system performance:

| Plan | Export Limit |
|------|--------------|
| **Standard** | 1,000 data points per export |
| **Advanced** | 10,000 data points per export |

For larger exports, use date range filters to break into multiple files.

---

## Using Exported Data

### In Spreadsheets
1. Open the CSV file in Excel or Google Sheets
2. Use filters and pivot tables for analysis
3. Create charts from sentiment or volume data
4. Compare across time periods

### In Data Tools
1. Import JSON into your analytics platform
2. Use date fields for time-series analysis
3. Aggregate by sentiment or source
4. Join with your internal data

### For Reporting
1. Export relevant date ranges
2. Calculate metrics externally
3. Build custom visualizations
4. Include in presentations or documents

---

## Tips

1. **Filter before export**: Narrow your search to get relevant data
2. **Use date ranges**: Smaller exports are faster and more manageable
3. **Save searches**: Re-run and export consistently over time
4. **Document your process**: Note filters used for reproducibility
